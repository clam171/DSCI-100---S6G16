1. Data Description

1. Players Data

This data was taken from a research group in Computer Science at UBC. They have been collecting data about how people play video games. This data has a list of all unique players, including data about each player.
The file contains the following information:

The table size is 196 rows x 7 columns

  • experience = 5 level (Beginner, Amateur, Regular, Pro, Veteran), type: character
  • subscribe = Whether they subscribe to game-related newsletter, type: logical
  • hashedEmail = unique identification for each player, type: character
  • played_hours = Amount of hours players play the game, type: double
  • name = Players' name, type: character
  • gender = Players' gender, type: character
  • Age = Players' age, type: double


2. Sessions Data

This data was also taken from a research group in Computer Science at UBC. This data has a list of individual play sessions by each player, including data about the session.
The file contains the following information:

The table size is 1535 rows x 5 columns

  • hashedEmail = unique identification for each player, type: character
  • start_time = Players' start time, type: character
  • end_time = Players' end time, type: character
  • original_start_time = UNIX start time, type: double
  • original_end_time = UNIX end time, type: double

2. Questions

Broad question:
What player characteristics and behaviours are most predictive of subscribing to a game-related newsletter, and how do these features differ between various player types?

Specific question:
Can played hours and age predict whether a player subscribe to a game-related newsletter in players dataset?

3. Exploratory Data Analysis and Visualization

I created 3 plots:

  1. Scatter plot of played hours vs Age

    2. The scatter plot does not show a strong pattern betweeen the explanatory variables and response variables. I notice that players who don't subscribe to game-related newsletter have below 10 hours of play time, age 20 and above. Players who subscribe have varied playing hours, higher play time players tend to subscribe, however, low play time players is more unpredictable.

  2. Histogram of played hours

    3. Both players who subscribe to newsletter or not have a play time of below 3 hours, a lot of them having 0 hours of play time.

  3. Histogram of Age

    4. Both players who subscribe to newsletter or not are between age 17 to under 30.

4. Methods and Plan

I plan to perform K-nearest neighbors classification to answer my specific question. This method is approriate because it can take 2 numeric variables to predict a categorical class.
Response variable: subscribe
Explanatory variables: played_hours, Age
Few assumptions are needed to apply the method.
The dataset is relatively small, which may limit the model's generalizability.
Here's a breakdown of the steps I would do:
  1. Centering and scaling, balancing the data by oversampling the rare class, drop missing data
  2. Split the data into 70% training set and 30% testing set
  3. Use 5-fold cross-validation to determine the optimal k value
  4. After tuning the K-NN classifier, evaluate the test set
  5. Assess the estimated accuracy of the test set, precision and recall, confusion matrix
